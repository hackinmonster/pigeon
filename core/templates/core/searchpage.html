Job Title: Front-End Developer Intern

Location: Charlotte, NC

 

Job Overview: We are looking for a motivated Front-End Developer Intern to assist in building and optimizing the user-facing components of our web applications. Key Responsibilities:

  • Assist in developing responsive web pages using HTML, CSS, and JavaScript.
  • Collaborate with designers to implement UI/UX designs.
  • Support front-end development for mobile-first web applications.
  • Debug and troubleshoot front-end issues.
  • Stay updated on emerging front-end technologies and best practices.
  • Qualifications:

 

Qualifications:

  • Basic understanding of HTML, CSS, and JavaScript.
  • Familiarity with front-end frameworks (e.g., React, Angular, or Vue.js) is a plus.
  • Eagerness to learn and adapt in a fast-paced environment.
  • Strong problem-solving skills and attention to detail.
